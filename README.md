# CarRental

Итоговый проект ШИФТ — сервис аренды машин. Пишу на SwiftUI, архитектура MVVM с координатором. Уровень Middle.

## Возможности

- Список машин с сервера, поиск по модели и фильтры (марка, тип кузова, руль, коробка, цена, цвет)
- Выбор периода аренды на главном экране с расчётом стоимости
- Экран машины: галерея фото, характеристики, итоговая цена за срок
- Оформление брони в 3 шага (даты и места → данные заказчика → проверка) и экран подтверждения
- Валидация полей на каждом шаге

## Стек

- Swift
- SwiftUI
- MVVM + Coordinator
- Сеть: URLSession + async/await
- Юнит-тесты на XCTest

## Сборка

Проект генерируется через XcodeGen, поэтому `.xcodeproj` не лежит в репозитории.

Ставим XcodeGen один раз:

    brew install xcodegen

Дальше всё через Makefile:

    make build

Проект сгенерируется и сразу откроется в Xcode — останется нажать «Run».

## Тесты

Логику покрываю юнит-тестами, запускаются из Xcode по Cmd+U.
